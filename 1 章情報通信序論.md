
# 1.0 情報通信序論 <a id="chap1"></a>
人類は今新しい時代を経験しつつあるといわれている。いわゆる高度情報化と呼ばれているものがそれである。高度情報化社会とは、物質あるいはエネルギー直野基礎的な資源に加えて、情報とその通信とが生活の基本的な部分に入り込み、人々の思考と行動とを決定する時代であるとされている。

現在の社会は、大量の情報の発生、迅速な加工とその伝達の手段の普遍化によって、大きな変革を経験しようとしている。現在生じている変革は、18世紀半ば以降に生じた物質あるいはエネルギーの加工・運搬手段の革新に比べて、はるかに大っきな衝撃を人類社会にもたらすものである。

本章では、情報通信、とりわけ電気的な情報の発生・加工・伝達の環境が、これまでどのような展開を見せたかを簡単に、情報通信の将来を展望する。
***

## 1.1 情報の伝達と記憶の歴史 <a id="chap1-1"></a>
人類は古くから、情報を伝達し、記憶し、共有する手段を発展させてきた。言語の成立は、知識や経験を他者へ伝えることを可能にし、集団の形成と文化の発展を支える基盤となった。しかし音声による伝達は時間と空間に制約されるため、情報を長期的に保存し、遠隔地へ届ける手段が求められた。

その制約を克服したのが文字の発明である。文字は情報を記録可能な形に固定し、世代を超えた伝承を可能にした。さらに印刷技術の発展は、情報の大量複製と広範囲への普及を実現し、社会構造や知識体系に大きな変化をもたらした。情報は特定の個人や集団に限定されるものではなく、社会全体に共有される資源へと変化した。

近代に入り、電気通信技術の登場は情報伝達の速度と距離の制約を大きく緩和した。電信、電話、無線通信などの発明により、情報は瞬時に遠隔地へ伝えられるようになり、社会活動の様式そのものが変化した。情報の流通量は飛躍的に増大し、通信は社会基盤の一部となった。

このように、情報の伝達と記憶の手段の発展は、人類社会の発展と密接に結びついている。情報通信技術は単なる技術的進歩ではなく、社会構造や経済活動、文化の在り方を変化させてきた重要な要素である。

## 1.2 電気通信の登場と展開の時代 <a id="chap1-2"></a>
情報を遠隔地へ迅速に伝達しようとする試みは古くから存在した。情報の記号化によって組織的な伝達を行った例としては、クロード・シャップによる腕木式通信（1793年）が挙げられる。しかし、情報伝達速度を飛躍的に向上させたのは電気通信の登場であった。

1835年、サミュエル・モールスが有線電信機を発明し、1844年には米国ボルチモア―ワシントン間で電信が実用化された。その後、電信網は急速に拡大し、19世紀末までには大西洋を越える世界規模の電信網が形成された。興味深いのは、1814年に登場した蒸気機関車による鉄道網よりも、電信網の方が短期間で世界的規模に発展した点である。これは情報伝達の需要がいかに大きかったかを示している。

1876年にはグラハム・ベルが電話機を発明し、1878年には手動電話交換業務が開始され、電話の時代が始まった。一方、1864年にはマクスウェルによって電磁波の存在が理論的に予言され、1887年にヘルツがその存在を実験的に確認した。さらにマルコーニは1895年に無線電信機を発明し、1901年には大西洋横断無線通信に成功するなど、無線通信も急速に発展した。

日本においても、1869年に東京―横浜間で電信業務が開始され、1890年には同区間で公衆電話交換業務が始まっている。無線通信も1903年には海軍に導入され、翌1904年には長崎―台湾間の通信に成功するなど、早期に展開した。また1925年にはラジオ放送が開始され、電気通信は広く社会に浸透していった。

電気通信が急速に発展した背景には、伝送距離の拡大と雑音の増大という技術的課題があった。長距離伝送では信号の減衰と雑音の蓄積が問題となるため、増幅技術と広帯域化が不可欠であった。20世紀初頭には三極真空管の発明（1906年）をはじめ、発振器、フィルタ理論、負帰還増幅器などの技術が進展し、伝送技術は飛躍的に向上した。これらの進歩が電話網や無線通信網の大規模化を可能にした。

このように電気通信は、単なる発明の積み重ねではなく、社会的要請と技術的課題の相互作用の中で急速に発展した。情報伝達の効率化は社会構造そのものを変化させ、今日の高度情報化社会の基盤を形成するに至っている。

## 1.3 電話網の成熟と伝送環境の変容 <a id="chap1-3"></a>
第二次世界大戦後、トランジスタや集積回路の登場により電子技術は飛躍的に進歩した。これに支えられて電気通信技術も高度化し、有線・無線双方において伝送容量は著しく拡大した。長距離伝送の品質と経済性は大きく向上し、電話網は社会インフラとして成熟段階に達した。日本においても[同軸ケーブル方式](https://ja.wikipedia.org/wiki/%E5%90%8C%E8%BB%B8%E3%82%B1%E3%83%BC%E3%83%96%E3%83%AB)やマイクロ波多重方式の導入が通信網の復旧と拡張に貢献し、1980年代には公衆電話網はほぼ完成を見るに至った。

しかし、電話網が成熟した後も伝送方式の革新は続いた。特に光通信方式の実用化は決定的であった。[半導体レーザ]()と光ファイバの発展により、従来の電話網が想定していた容量を大きく上回る伝送能力が実現された。また、衛星通信や無線通信の進展は、固定的な有線網とは異なる柔軟な通信環境を生み出した。

これまで約100年間、電気通信網は電話を中心として設計されてきた。電話帯域（約4kHz）を基準とするアナログ通信網が構築され、デジタル化後も標本化周期125μs、8ビット量子化、64kbpsを基本単位とする設計思想が維持された。しかし、伝送容量の飛躍的増大により、電話中心の枠組みは相対化され、映像や高精細情報の伝送といった新たな需要が顕在化し始めた。こうして通信環境は、電話網中心の時代から大容量情報通信の時代へと変容していった。

## 1.4 情報通信網への展開 <a id="chap1-4"></a>

前節までに見てきたように、電気通信技術はほぼ1世紀をかけて世界規模の電話通信網を構築し、成熟段階に達した。しかしその一方で、新しい世紀を目前にして、電気通信技術はかつて経験したことのない状況に直面している。それが「増幅と広帯域化という技術的制約の解消」である。かつてエンジニアたちが取り組んできた課題——長距離伝送での信号劣化や雑音への対処——は、1.3節で見たように半導体技術と光通信の進歩によって実質的に解決された。つまり「伝送できる量が足りない」という前提が崩れたのである。こうして電気通信技術は今、従来の枠組みを超えた、まったく新しい視点からの問い直しを迫られている。

この変化をもたらした最大の要因は、[半導体集積化技術](https://ja.wikipedia.org/wiki/%E9%9B%86%E7%A9%8D%E5%9B%9E%E8%B7%AF)の目覚ましい進展である。半導体技術の進歩は単に通信の手段を変えたのではなく、通信の「目的」そのものを変容させた。情報処理技術の飛躍的な向上により、通信は「ヒトとヒトをつなぐ」手段だけではなく、「機械と機械をつなぐ」基盤へと変化しつつある。

まず、これまでの情報通信がどのような前提に基づいていたかを整理しよう。1.3節で触れたように、電話網は音声帯域（約4kHz）を基準に設計されており、デジタル化後も64kbpsを基本単位とする設計が維持されてきた。この枠組みのもとで、情報通信の主要な課題は次の2点に集約されていた。

- **[情報源符号化](https://ja.wikipedia.org/wiki/%E7%AC%A6%E5%8F%B7%E5%8C%96)（帯域圧縮）**：狭い伝送路にできるだけ多くの情報を詰め込む技術。アナログ信号から発生する情報を、限られた伝送容量に合わせるための圧縮手法が研究の中心であった。
- **[通信路符号化](https://ja.wikipedia.org/wiki/%E8%AA%A4%E3%82%8A%E8%A8%82%E6%AD%A3%E7%AC%A6%E5%8F%B7)（誤り訂正）**：伝送中に生じる誤りを検出・訂正する技術。送受信端末に大きな記憶容量が持てないという前提のもとで、原信号を「瞬時に・忠実に」伝達することが至上命題とされていたためである。

しかし半導体技術の進歩により、端末（スマートフォンやPCなど）は大容量のメモリと高い演算能力を持つようになった。これは情報通信の前提を根本から変える。「信号を瞬時に忠実に届ける」必要はもはやない。音声・静止画像・動画像といった情報は、**送信前に端末側で加工・蓄積し、最適なタイミングと形式で伝送できる**ようになったのである。たとえば電話音声でさえ、ヒトが発声する情報量は時間的に一定ではない。従来はこの変動を無視して固定レートで送っていたが、新しい考え方ではこの統計的性質を積極的に利用することで、同じ回線でより高品質な伝達が実現できる。さらに、音声・静止画像・動画像を同じ伝送路で複合的に扱う、いわゆる**[マルチメディア](https://ja.wikipedia.org/wiki/%E3%83%9E%E3%83%AB%E3%83%81%E3%83%A1%E3%83%87%E3%82%A3%E3%82%A2)通信の実現も視野に入ってきた。これは、音声専用に最適化された従来の伝送方式を根本的に見直すことを求めるものである。

光ファイバ通信の実用化により、経済的に実現できる伝送能力はほぼ無制限ともいえる状況に達しつつある。この状況がもたらす変化は革命的である。従来の通信は「ヒトとヒトを直接つなぐ」ことを前提としていたが、これからの通信では**情報処理機械（コンピュータやAIなど）が人間の代わりに通信を行う**という姿が現れてくる。個人の秘書役を担う情報処理機械がユーザーに代わって情報を自動収集・整理・伝達し、機械同士が膨大な量の通信を行ってその結果だけをヒトに提示する——このような未来像は、今日のAIアシスタントやクラウドサービスにすでにその萌芽を見ることができる。

一方で、このような新しい通信環境には新たな技術的課題も生じる。その一つが**接続遅延**の問題である。電話網では、ヒトが会話する上で許容できる遅延（数百ミリ秒程度）を前提に設計されており、接続に数百msかかることが許容されていた。しかしGbps（ギガビット毎秒）規模の通信速度が実現された世界では、情報処理機械にとってこの遅延は致命的であり、電話網の設計思想をそのまま引き継ぐことはできない。

このような技術的転換の時代において、情報通信技術者に求められることについて、著者は明快にこう述べる。それは**あらゆる既成の概念の枠を固定化せず、柔軟な発想を持ち、対象を自然に把握すること**である。電気通信技術はこれまで、増幅・広帯域化という技術的な壁に挑み、その壁を破ることで発展してきた。現在の通信システムは巨大な世界的インフラとして成長しており、過去の投資の蓄積がある意味で制約ともなっている。しかしだからこそ、若い技術者が「改良的な視点だけに固執する」のではなく、遠い将来を見通した構想力を持つことに大きな意義がある。

本節は第1章「情報通信序論」の締めくくりであり、本書全体を通じて何を学ぶべきかを示した節でもある。情報の蓄積・処理・伝達は古くかつ新しい課題であり、情報処理技術と情報通信技術の融合によってヒトに優しい情報環境を作り上げることこそが、技術者の究極の目標である。情報通信の「技術的原理」を学ぶことは、単なる知識の習得ではなく、新しい時代を切り拓くための思考の基盤を作ることに他ならない。
