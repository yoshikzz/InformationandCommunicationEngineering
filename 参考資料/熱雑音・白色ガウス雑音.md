## 第1章：電気の基本

### 電子とは？

物質の中には**電子**という非常に小さな粒が無数に存在しています。

```
電線の中：
[ ● ● ● ● ● ● ● ● ● ● ]
  電子がぎっしり詰まっている
```

### 電流・電圧とは？

```
電子が一方向に流れる
        ↓
これが「電流」

電子を押し出す力
        ↓
これが「電圧」

💧 水のイメージ：
  水圧（電圧）→ 水の流れ（電流）→ 水の粒（電子）
```

---

## 第2章：電圧は「揺れる」ことがある

電圧はずっと一定とは限りません。**上下に揺れる**ことがあります。

```
ずっと一定の電圧（乾電池）：
電圧
↑
│―――――――――――――――――――――
└──────────────────────→ 時間


上下に揺れる電圧（コンセントなど）：
電圧
↑    ／＼        ／＼
│   /    \      /    \
│──/──────\────/──────\──→ 時間
│          \  /
│            \/
```

---

## 第3章：周波数とは「揺れの速さ」

電圧が**1秒間に何回 上下に揺れるか**を「周波数」といいます。 単位は**Hz（ヘルツ）**です。

```
ゆっくりした揺れ（低周波数）：
電圧
↑   ／―――＼            ／
│  /         \          /
│ /            \________/
└──────────────────────→ 時間
  1秒間に数回しか揺れない


速い揺れ（高周波数）：
電圧
↑ /\/\/\/\/\/\/\/\/\/\/\/\
│
└──────────────────────→ 時間
  1秒間に何百・何万回も揺れる
```

### 身近な例

|周波数|具体例|
|---|---|
|50Hz|家のコンセントの電気|
|440Hz|ピアノの「ラ」の音|
|20,000Hz|人間の聞こえる限界|
|2,400,000,000Hz|Wi-Fi|

---

## 第4章：複数の周波数が混ざると複雑な波になる

異なる速さの揺れを**足し合わせる**と複雑な波形になります。

```
ゆっくりした揺れ（低周波）：
↑  ／――＼      ／――＼
│ /       \    /
│/         \__/
└─────────────────────────→

速い揺れ（高周波）：
↑ /\/\/\/\/\/\/\/\/\/\/\/\/
│
└─────────────────────────→

2つを足し合わせると：
↑  ╭╮╭╮╭╮╭╮  ╭╮╭╮╭╮╭╮
│ ╭╯╰╯╰╯╰╯╰╮╭╯╰╯╰╯╰╯╰╮
│╭╯          ╰╯
└─────────────────────────→

→ 複雑な波形になる！
```

逆に複雑な波形も**周波数ごとに分解して**、どの周波数がどのくらい含まれているか調べることができます。

```
複雑な波形を分解すると…

パワー
↑        ■
│      ■ ■ ■
│    ■ ■ ■ ■ ■
│  ■ ■ ■ ■ ■ ■ ■
└──────────────────→ 周波数
  低い            高い

→ どの周波数がどのくらい含まれているか一目でわかる！
```

## 第5章：熱運動とは何か

ここからがノイズの話です。

物質の中の電子は、温度がある限り**常にランダムに動き回っています**。これを「熱運動」といいます。

```
温度が低い：
[ ●  ●  ●  ●  ●  ●  ● ]
  あまり動かない、おとなしい


温度が高い：
[ ●↗ ↙● ●↑ ↓● ●↘ ↖● ]
  激しくバラバラに動き回る！
```

> 💡 温度が高いほど電子は激しく動く

---

## 第6章：熱運動がノイズを生む（熱雑音）

電子がランダムに動き回ると、**電圧がランダムにガタガタ揺れます**。

```
熱運動による電圧の揺れ：

電圧
↑  ╭╮    ╭─╮  ╭╮
│ ╭╯╰╮  ╭╯  ╰──╯╰╮  ╭╮
│─╯  ╰──╯        ╰──╯╰─
└────────────────────→ 時間

→ まったく規則性のないガタガタした揺れ
→ これが「熱雑音」！
```

### 重要なポイント

```
・抵抗など電子回路のあらゆる部品で必ず発生する
・絶対に消すことができない
・温度が高いほどノイズが大きくなる

温度と熱雑音の関係：
絶対零度（-273℃）→ ノイズなし（電子が動かない）
室温（27℃）      → 普通のノイズ
高温（227℃）     → ノイズが大きい
```

## 第7章：「白色」の意味

この熱雑音のガタガタ波形を**周波数ごとに分解**して調べると…

```
パワー
↑
│ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■
│ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■
│ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■
└──────────────────────→ 周波数
  低い                高い

→ 全部の周波数が均等！
```

これを**「白色」** と呼びます。光と同じ理由です👇

```
🔴赤＋🟠橙＋🟡黄＋🟢緑＋🔵青
    ↓ 全ての色（周波数）を均等に混ぜると
        ⚪ 白色光！

低周波＋中周波＋高周波
    ↓ 全ての周波数を均等に含むと
        ⚪ 白色雑音！
```

### 音で例えると

```
ホワイトノイズ（ザーッという音）：
  低い音の成分：■■■■
  中くらいの音：■■■■
  高い音の成分：■■■■
  → 全部均等！→「ザーッ」という音

低音ばかりのノイズ：
  低い音の成分：■■■■■■■■
  中くらいの音：■■
  高い音の成分：■
  → 偏っている！→「ボーッ」という音
```

## 第8章：「ガウス」の意味

熱雑音の電圧の値を**ひたすら記録**して、どの値が何回出たかグラフにすると…

```
記録した値：
0.1V, -0.3V, 0.0V, 0.2V, -0.1V, 0.0V...

頻度
↑
│          ██
│        ██████
│      ██████████
│    ██████████████
│  ██████████████████
│████████████████████████
└────────────────────────→ 電圧値
     マイナス  0V  プラス

→ 0V付近が一番多い
→ 大きな値になるほど稀
→ 釣り鐘型！＝ガウス分布！
```

### なぜ釣り鐘型になるの？

```
無数の電子がそれぞれランダムに動く
        ↓
全部足し合わさると…
        ↓
「0付近が一番多い」釣り鐘型に自然となる

サイコロで例えると：
サイコロ1個    → 1〜6が均等に出る
サイコロ10個の合計 → 真ん中の値が一番出やすい！
        ↓
電子も同じ！無数に集まると釣り鐘型になる
```

---

## 第9章：全部つなげる

```
① 電子は温度がある限り常にランダムに動き回る
            ↓
② 電圧がランダムにガタガタ揺れる（熱雑音発生）
            ↓
③ そのガタガタを周波数ごとに分解すると
  低い周波数も高い周波数も全部均等に含まれている
            ↓
  これを「白色」と呼ぶ
            ↓
④ そのガタガタの電圧値の出方を記録すると
  0V付近が多く、大きな値は稀な釣り鐘型になる
            ↓
  これを「ガウス分布」と呼ぶ
            ↓
⑤ 「白色」＋「ガウス」＝「白色ガウス雑音」
```

## 🎯 最終まとめ

|言葉|意味|
|---|---|
|**電子の熱運動**|温度がある限り電子が常にランダムに動き回ること|
|**熱雑音**|熱運動によって電圧がガタガタ揺れる現象|
|**白色**|低周波から高周波まで全ての揺れが均等に含まれること|
|**ガウス**|電圧の値の出方が釣り鐘型（0付近が多く大きい値は稀）になること|
|**白色ガウス雑音**|上記の性質を全て持つ、熱雑音の正体|
|**S/N比**|この雑音に対して、目的の信号がどれだけ大きいかの比率|
