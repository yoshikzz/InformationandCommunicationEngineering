

## まず「多重通信」とは何か？

どちらの方式も「**多重通信**」を行っています。まずこの概念を理解しましょう。

```
【多重通信のイメージ】

昔（1回線 = 1通話）：
  東京 ━━━━━━━━━━━━━ 大阪
         1本の線 = 1通話のみ

多重通信（1回線 = 何千通話）：
  東京 ━━━━━━━━━━━━━ 大阪
         1本の線に何千もの通話を同時に乗せる！
```

> 💡 **多重通信のメリット：** 1本のケーブルや1本の電波で、同時に何百・何千もの電話をかけられる！インフラのコストを劇的に削減できます。

# 🔵 ① 同軸ケーブル方式

## 同軸ケーブルとは？

まず「同軸ケーブル」の構造から理解しましょう。テレビのアンテナケーブルがまさにこれです。

```
【同軸ケーブルの断面構造】

        ┌────────────────────────────┐
        │  ④ 外部被覆（プラスチック）  │ ← 外側の保護
        │  ┌──────────────────────┐  │
        │  │  ③ 外部導体（網状の金属）│  │ ← 電磁シールド
        │  │  ┌────────────────┐  │  │
        │  │  │  ② 絶縁体       │  │  │ ← 内外を絶縁
        │  │  │  ┌──────────┐  │  │  │
        │  │  │  │ ① 中心導体 │  │  │  │ ← 信号が通る
        │  │  │  │  （銅線）  │  │  │  │
        │  │  │  └──────────┘  │  │  │
        │  │  └────────────────┘  │  │
        │  └──────────────────────┘  │
        └────────────────────────────┘
        
        名前の由来：中心導体と外部導体が
        同じ「軸」の上にある → 「同軸」ケーブル！
```

### 普通の電話線との違い

||普通の電話線（ツイストペア）|同軸ケーブル|
|---|---|---|
|**構造**|2本の銅線をより合わせた線|同心円状の4層構造|
|**伝送帯域**|数kHz（電話1本分）|数十MHz（数千本分）|
|**ノイズ耐性**|弱い|**非常に強い**（外部導体がシールド）|
|**伝送距離**|短い|長い|

---

## 同軸ケーブルで多重通信する仕組み：FDM（周波数分割多重）

同軸ケーブルの広い帯域を活かして、**複数の通話を異なる周波数に割り当てて同時送信**します。

```
【FDM（周波数分割多重）のイメージ】

周波数
  ↑
  │ ┌──────┐
  │ │通話A  │  0.3〜4 kHz（基本帯域）
  │ └──────┘
  │          ┌──────┐
  │          │通話B  │  64〜68 kHz
  │          └──────┘
  │                    ┌──────┐
  │                    │通話C  │  68〜72 kHz
  │                    └──────┘
  │                              ┌──────┐
  │                              │通話D  │  72〜76 kHz
  │                              └──────┘
  │                    ・・・・・・（何千本も）
  └─────────────────────────────────────→ 帯域幅

   ↑同軸ケーブル1本でこれだけの通話を同時に運ぶ！
```

> 💡 **例え話：** 高速道路のようなもの。1本の道路（同軸ケーブル）に複数の車線（周波数帯）を作り、それぞれの車線で別の車（通話）を走らせるイメージです。

### 日本での実用例

|時代|方式|伝送容量（電話回線数）|
|---|---|---|
|1959年|東京〜大阪間 同軸ケーブル敷設|**960チャネル（回線）**|
|1970年代|60MHz方式|**10,800チャネル**|

```
【伝送容量の拡大イメージ】

従来の電話線（1本 = 1通話）：
 ■

同軸ケーブル初期（960回線）：
 ■■■■■■■■■■■■■■■■■■■■ × 48行（960個）

60MHz方式（10,800回線）：
 桁違いの大容量！！
```

# 🔴 ② マイクロ波多重方式

## マイクロ波とは？

```
【電波の周波数帯と種類】

長波     中波     短波    超短波    マイクロ波         ミリ波
 ↓        ↓        ↓       ↓    ┌────────────┐      ↓
─────────────────────────────────│300MHz〜300GHz│────────
                                 └────────────┘
                                 ← ここがマイクロ波
                                 
                                 波長：1mm 〜 1m
                                 「マイクロ（微小）な波長」が名前の由来
```

|種類|周波数|波長|特徴|
|---|---|---|---|
|AM ラジオ|約1 MHz|300m|山や建物を回り込む|
|FM ラジオ|約100 MHz|3m|ある程度回り込む|
|**マイクロ波**|**2〜15 GHz**|**2〜15cm**|**直進性が強い**|
|光|数百 THz|数百nm|完全に直進|

---

## マイクロ波通信の仕組み：見通し中継

マイクロ波は**光のようにまっすぐ直進する性質**があります。そのため「見通し距離」でしか届きません。

```
【マイクロ波中継の仕組み（全国中継ネットワーク）】

 東京         中継所①        中継所②        大阪
  ┃           ┃               ┃              ┃
  ▲  )))→→→  ▲   )))→→→    ▲  )))→→→    ▲
  鉄 ←(((    鉄  ←(((       鉄  ←(((      鉄
  塔           塔              塔              塔
（山の上など見晴らしの良い場所に設置）

← 約50km → ← 約50km → ← 約50km →

・各中継所でいったん受信して増幅し、再送信する
・これを繰り返して全国をつなぐ
```

> 💡 **なぜ鉄塔が山の上にあるのか？** マイクロ波は直進するため、途中に山や建物があると遮断されてしまいます。そのため見晴らしの良い高い場所に中継アンテナを設置する必要があります。

```
【マイクロ波多重通信のフロー】

  ①複数の通話                ②マイクロ波に         ③空中を伝搬
  ┌──────┐                  変換して送信
  │通話A  │ ─┐               ┌─────────┐
  ├──────┤  │               │         │   )))))))→→→
  │通話B  │ ─┤→[FDM合成]→   │送信機   │  アンテナから
  ├──────┤  │               │（鉄塔）  │  空間へ放射
  │通話C  │ ─┤               └─────────┘
  ├──────┤  │
  │  ：   │ ─┘
  └──────┘

          ④中継所で受信・再送信
          ↓↓↓
  ⑤次の中継所または受信局へ
```

# 📡 なぜ同軸ケーブルは広い帯域幅を持つのか？

---

## まず「帯域幅」とは何か？

```
【帯域幅のイメージ】

         使える周波数の範囲
         ←───────────────→
         ┌─────────────────┐
周波数   │  ここが「帯域幅」  │
─────────┴─────────────────┴──→
       低い                 高い

帯域幅が広い = 一度に運べる情報量が多い
帯域幅が狭い = 一度に運べる情報量が少ない
```

> 💡 **道路に例えると：** 帯域幅 ＝ 道路の幅。広い道路ほど多くの車（情報）が同時に走れます。

---

## 普通の電話線（ツイストペア）の問題点

普通の銅線2本をよじった電話線は、**高い周波数になるほど信号が弱まってしまいます（減衰）**。

```
【普通の電話線で高周波を送るとどうなるか】

信号の強さ
  ↑
強│●
  │ ●
  │   ●
  │     ●●
  │         ●●●
弱│               ●●●●●●●●→ 消えてしまう！
  └──────────────────────────→ 周波数
    低い（使える）   高い（使えない）
    
→ 実用的に使えるのは数kHzまで（電話1本分だけ）
```

### なぜ高周波だと減衰するのか？「表皮効果」

```
【表皮効果のイメージ】

【低周波（電流が銅線全体を流れる）】

     ┌─────────────────┐
     │ ← ← ← ← ← ←  │  ← 断面全体に
     │ ← ← ← ← ← ←  │    電流が流れる
     │ ← ← ← ← ← ←  │
     └─────────────────┘
     導線の断面          → 抵抗が小さい ✅

【高周波（電流が表面しか流れない）】

     ┌─────────────────┐
     │                 │
     │      空洞！      │  ← 中心は使われない
     │                 │
     │← ←           →→│  ← 表面だけに電流集中
     └─────────────────┘
     導線の断面          → 実効的な断面積が激減
                         → 抵抗が大きくなる
                         → 信号が熱に変わって減衰 ❌
```

> 💡 **表皮効果：** 高周波になるほど電流が導線の「表面（皮膚）」だけに集中し、中心部は使われなくなる現象。周波数が高いほど影響が大きくなります。

## 同軸ケーブルが広い帯域幅を持てる理由

同軸ケーブルには、帯域幅を広げる**3つの工夫**があります。

---

### 理由① 外部導体がノイズを完全にシャットアウト

```
【外部からのノイズ（電磁干渉）への耐性】

普通の電話線：
                ⚡ノイズ
                  ↓
    ────────────────────────→  ← 信号線
    ⚡ノイズが容赦なく侵入！
    ⚡ → 高周波信号ほど影響を受けやすい
    
同軸ケーブル：
                ⚡ノイズ
                  ↓
    ┌────────────────────────┐
    │③外部導体（金属シールド）  │← ノイズをここで遮断！
    │  ┌────────────────┐    │
    │  │②絶縁体          │    │
    │  │  ┌──────────┐  │    │
    │  │  │①中心導体   │  │    │← 信号がここを通る
    │  │  └──────────┘  │    │   ノイズが届かない✅
    │  └────────────────┘    │
    └────────────────────────┘
    
→ 高周波信号でも外部ノイズに影響されず安定して伝送できる！
```

---

### 理由② 外部導体が「電磁波の漏れ」も防ぐ

```
【高周波信号と電磁波の放射】

普通の電話線（高周波を流すと）：
    ─────────────────────────→
         ↗↗↗ 電磁波が外に漏れる！
    エネルギーが失われる → 信号が弱まる ❌
    また外部にも電磁波を放射してしまう（他の通信に干渉）

同軸ケーブル（高周波を流しても）：
    ┌────────────────────────┐
    │③外部導体                 │← 電磁波を内部に閉じ込める！
    │  ┌────────────────┐    │
    │  │  ①中心導体      │    │← 電磁波が外に漏れない
    │  └────────────────┘    │   エネルギー損失が少ない ✅
    └────────────────────────┘
    
→ 高い周波数でも信号エネルギーが外に逃げない！
```

---

### 理由③ インピーダンスが均一 → 信号の反射・歪みが起きない

```
【インピーダンス（信号の通りにくさ）の均一性】

普通の電話線：
    ─────────────────────────→
    環境によってインピーダンスが変わる
    → 信号が途中で反射・歪む
    → 高周波ほど影響が大きい ❌

同軸ケーブル：
    中心導体と外部導体の距離が
    どこでも完全に一定
         ↓
    インピーダンスが均一！
         ↓
    信号の反射・歪みが起きない ✅
    → 高周波でも安定して伝送できる
```

---
